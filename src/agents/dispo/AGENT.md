# Dispo Agent

**Domain:** Action core
**Trigger:** Deal goes under contract
**Output:** Ranked buyer-fit list, outreach drafts
**Review gate:** Operator selects buyer and approves outreach
**Utilization tier:** Tier 3 (compounding, needs buyer memory and deal flow)

## MCP tools used

- Sentinel MCP: deal context, buyer profiles, buyer history
- Composio MCP: Gmail (outreach drafts), Calendar (showing scheduling)
